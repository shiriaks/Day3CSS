HTML

Hyper Text Markup Language

HTML elements are the building blocks of HTML pages.
With HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page.
HTML provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items.

Most graphical email clients allow the use of a subset of HTML (often ill-defined) to provide formatting
and semantic markup not available with plain text.
This may include typographic information like coloured headings, emphasized and quoted text, inline images and diagrams.
